中国的金融体系

中国金融市场的历史、现状及影响因素

新中国成立以来我国金融市场的发展历史

  1. 1949-1978：计划经济时期大一统的金融体系
    • 建立新中国金融体制的两个首要任务：建立统一的人民币制度和成立新中国的金融机构体系。
    • 1948年12月1日，中国人民银行成立，第一套人民币同时发行。
    • 1952年12月，全国统一的公私合营银行成立，后并入中国人民银行体系。
    • “大”是指中国人民银行分支机构覆盖全国；
    • “一”是指中国人民银行是该时期唯一的银行，它集中央银行和商业银行双重职能于一身，集现金中心、结算中心和信贷中心于一体；
    • “统”是指全国实行统存统贷的信贷资金管理体制。
  2. 1979-1983：多元混合型金融体系。中国人民银行仍然身兼二职，既承担中央银行职能，又办理信贷储蓄业务。
  3. 1984-1993：以中央银行为中心，国家专业银行为主体，多种金融机构分工协作的金融体系逐步形成。
    • 1983年9月，中国人民银行开始专门行使中央银行职能
    • 1992年10月，国务院证券委员会和中国证券监督管理委员会成立，标志着中国资本市场开始逐步纳入全国统一监管框架，全国性市场由此形成并初步发展。
  4. 1994-2003：逐步改革和完善的市场金融体系。金融体制改革的重点：
    • 要把中国人民银行办成真正的中央银行，主要用经济办法调控货币供应量，保证币值稳定；
    • 现有专业银行要逐步转变为商业银行；
    • 组建政策性银行，实行政策性业务与商业性业务分开；
    • 改革外汇管理体制，建立以市场为基础的、有管理的浮动汇率制度和统一规范的外汇市场。
  5. 2004-2011：多种所有制和多种经营形式、结构合理、功能完善、高效安全的现代金融体系。分业经营、分业监管
  6. 2012年至今：基本金融制度逐步健全、对外开放程度不断提高的现代金融体系。人民币国际化和金融业双向开放促进了金融体系的不断完善。

我国金融市场的发展现状

货币市场

我国货币市场建设始于1984年银行间同业拆借市场的建立。目前已经形成包括银行间同业拆借市场、短期债券市场、债券回购市场和票据贴现市场在内的统一的货币市场格局。

债券市场

债券交易市场：

  • 场外市场
    • 银行间市场：参与者限定为机构。场外债券批发市场
    • 银行柜台市场：参与者限定为个人。场外债券零售市场
  • 场内市场
    • 上海证券交易所
    • 深圳证券交易所

我国债券市场发行债券：

  1. 利率债：直接以政府信用为基础或是以政府提供偿债支持为基础而发行的债券。
    • 国债、地方政府债、中央银行发行的票据等
  2. 信用债：以企业的商业信用为基础而发行的债券。
    *中期票据、短期融资券、企业债、公司债

股票市场

  • 1984年11月18日改革开放以来第一只公开发行的股票飞乐音响诞生；
  • 20世纪90年代开始发行B股；
  • 2002年引入合格境外投资者（QFII），后逐步扩大至人民币合格境外投资者（RQFII）；
  • 2014年11月17日，沪港通股票交易正式启动；
  • 2016年12月5日，深港通正式启动。

外汇市场

人民币外汇市场：在岸（CNY）市场、离岸（CNH）市场

随着我国外汇管理体制改革和汇率形成机制的完善，初步形成了外汇零售和银行间批发市场相结合，竞价和询价交易方式相补充，覆盖即期、远期和掉期等类型外汇交易工具的市场体系。

黄金市场

2002年10月，上海黄金交易所正式运行，实现了中国黄金生产、消费、流通体制的市场化，标志着我国国内黄金市场的正式开放。中国黄金市场目前初步形成了以上海黄金交易所黄金现货为主体，上海期货交易所黄金期货、商业银行柜台黄金产品共同组成的，层次分明且各有侧重的多元市场体系。

特点：

  • 市场架构比较完整；
  • 参与主体类型多样；
  • 与实体经济和黄金产业密切关联；
  • 投资与风险分散功能进一步得到发挥。

商品期货市场

  • 1990年10月我国第一家商品期货交易所，郑州商品交易所前身郑州粮食批发市场成立。
  • 2021年4月19日，广州期货交易所揭牌仪式举行；
  • 目前，我国成立了上海期货交易所、大连商品交易所、郑州商品交易所和广州期货交易所四大商品期货交易所。

金融衍生品市场

  • 2006年9月，中国金融期货交易所正式成立，成为我国首家金融衍生品交易所。
  • 2010年4月16日，中国金融期货交易所退出沪深300股指期货合约，标志着我国金融衍生品市场进入稳步发展阶段。
  • 2015年2月9日，我国首个场内金融齐全，上证50ETF期权，在上海证券交易所正式上市交易。
  1. 利率类衍生品
    • 场内国债期货
    • 场外利率远期
    • 场内利率互换产品
  2. 权益类衍生品
    • 股票期权
    • 股指期货
    • 认股权证
  3. 货币类衍生品
    • 人民币外汇期权和期权组合
    • 外汇远期
    • 外汇掉期
    • 货币掉期（互换）
  4. 信用类衍生品
    • 信用风险缓释合约
    • 信用风险缓释凭证
    • 信用违约互换
    • 信用联结票据

影响我国金融市场运行的主要因素

经济因素（最重要）

  • 经济增长
  • 经济周期波动
  • 宏观经济政策
  • 特定的财政金融行为
  • 国际经济环境

宏观经济运行对金融市场的影响

影响金融市场运行的主要宏观经济变量：

  1. 国内生产总值：GDP是衡量一国经济状况的最佳指标。
  • GDP增速加快，表明经济处于扩张阶段，经济增长强劲；
  • 持续、稳定、高速的GDP增长，通常意味着社会总需求与总供给协调增长，经济结构趋于合理，经济发展势头良好。
  1. 经济周期：金融市场运行受到经济所处不同周期的影响，金融市场随着经济周期的复苏、繁荣、衰退、萧条等不同阶段呈现出同期甚至超前的变化，金融市场是经济周期的直接构成者和最为敏感的反射物。
  2. 通货膨胀与利率：通货膨胀速度对金融市场投融资产生重要影响。利率对金融市场的影响是系统性、全局性的。
  • 低于预期的通货膨胀会降低利率，促进债券和股票价格的上升。
  • 高于预期的通货膨胀会提高利率，从而导致债券和股票价格的下降。
  • 利率下降时，企业利息负担降低，利润增加，反映在金融市场上就是既增加了金融资产的基础价值，也大大提高了投资者的获利预期，从而提高金融资产价格。但过低的利率也导致过度信贷杠杆，大幅推高资产价格给金融市场造成冲击。
  • 利率上升则会带来相反的影响。
  1. 汇率和国际资本流动
  • 国际资本从发达国家流向新兴市场国家，有利于后者经济发展和金融市场发展；
  • 国际资本流动造成了外汇市场和汇率的波动；
  • 国际资本流动造成借贷市场和利率的波动；
  • 当国际资本以证券投资形式流入国内，会刺激对国内的需求，引起证券价格上升；反之，相反。

宏观经济政策

  1. 货币政策：对金融市场有着直接的、主要的影响。货币政策主要通过货币供给量、利率和信贷政策机制发生作用。
  2. 财政政策：财政政策主要通过税收政策、公共支出政策以及国债发行等发挥作用。
  3. 收入政策：收入政策具有更高层次的调节功能，它制约着财政政策和货币政策的作用方向和作用力度，而且收入政策最终也要通过财政政策和货币政策来实现。收入政策目标包括收入总量目标和收入结构目标。

国际经济环境因素

  1. 经济全球化
    • 国际金融市场一体化加快了金融信息传递的速率，扩大了金融信息传递的范围，提高了金融市场效率；
    • 全球金融市场的关联性增强，增强了国际资本的流动性，各国宏观政策的外溢效应更加明显，由国际金融市场风险引发国内金融市场动荡的可能性也在增加。
  2. 放松管制和加强管制两种经济哲学的交替
  3. 世界货币制度的影响

法律因素

  1. 法律制度的完善与否、执法效率的高低等因素对金融市场运行有着重要影响；
  2. 完备的金融法律和高效的执法效率是金融市场平稳运行的根本保障。

市场因素

  1. 国内和国际统一的市场组织
  2. 丰富的市场交易品种
  3. 市场交易机制或市场模式：合理的定价机制，灵活的价格机制。

技术因素

科技发展对金融市场运行有重大意义。以信息技术为例：

  1. 信息技术有力地刺激了金融创新；
  2. 信息技术使信息披露更加及时，增强了金融机构的透明性，改变了金融监管的方式；
  3. 信息技术的广泛应用促使全球统一金融市场的形成，国际金融与国内金融的界限日益模糊。

心理因素

市场参与者的心理和行为可以加剧金融风险。

体制和管理因素

政府对金融市场的适当干预仍可能是必要的。完善的金融体制和金融监管能够通过行业规划、政策引导、市场监管、信息披露以及有序的进入与退出机制等手段，影响金融市场发展的规范与秩序。

其他因素

文化因素、人口因素

中国的金融中介机构体系

我国金融中介机构体系的构成

金融中介机构的定义

  • 狭义的金融中介机构一般指商业银行。
  • 广义的金融中介机构除商业银行以外，还有许多非银行金融机构（证券、保险、基金公司），它们从事综合的或专门的金融业务和金融股服务，形成相互联系、相互影响的统一整体，构成金融中介机构体系。

金融中介机构的分类

  • 业务特性及货币创造、交换媒介和支付手段的能力：银行金融中介机构和非银行金融中介机构
  • 是否承担国家政策性金融业务：商业性金融机构和政策性金融机构
  • 所从事业务的性质及其在金融活动中所起的作用：融资类金融中介机构、投资类金融中介机构。保障类金融中介机构、服务类金融中介机构

我国的金融中介机构体系

  • 银行机构体系：以中央银行为主导、国有商业银行为主体
  • 投资中介体系：以证券公司、期货公司和证券投资基金为主，以投资咨询中介、信托机构为辅
  • 保险中介体系

主要金融中介机构的业务

商业银行的业务

  1. 负债业务
    • 自有资金
    • 存款负债：一般占负债总额的**70%**以上
    • 借款负债
  2. 资产业务
    • 贷款业务
    • 投资业务
    • 现金资产
    • 票据贴现
  3. 表外业务
    • 担保承诺类
    • 代理投融资服务类
    • 中介服务类

证券公司的业务

  • 证券市场投融资服务的提供者
  • 证券市场重要的机构投资者
  1. 证券经济业务
  2. 证券投资咨询业务
  3. 与证券交易、证券投资活动有关的财务顾问业务
  4. 证券承销与保荐业务
  5. 证券自营业务
  6. 证券资产管理业务
  7. 融资融券业务
  8. 证券公司中间介绍业务
  9. 私募投资基金业务和另类投资服务等

保险公司的业务

财产保险公司业务：

  • 基础类业务
    • 机动车保险
    • 企业/家庭财产保险及工程保险
    • 责任保险
    • 船舶/货运保险
    • 短期健康/意外伤害保险
  • 扩展类业务
    • 农业保险
    • 特殊风险保险
    • 信用保证保险
    • 投资型保险

人身保险公司业务：

  • 基础类业务
    • 普通型保险
    • 健康保险
    • 意外伤害保险
    • 分红型保险
    • 万能型保险
  • 扩展类业务
    • 投资连结型保险
    • 变额年金

我国金融业主要行业的基本情况

  • 证券业
  • 保险业
  • 信托业

银行业

  • 大型商业银行：在资产规模、资金来源和网点布局上占据主导地位，业务遍布全国主要城市及大部分县城地区，网点众多且分散，各项业务均衡发展。
  • 股份制商业银行：主要业务集中在全国大中型城市及少数县域地区。
  • 城市商业银行：通常在特定的区域从事商业银行业务，依靠地缘优势、当地经济发展水平，重点针对当地企业和居民的需求提供金融产品和服务。

我国金融市场的监管体制

我国金融监管体制的演变

  • 1992年以前：中国人民银行统一监管阶段
  • 1992-1997年：分业监管起步阶段
  • 1998-2008：“一行三会”体制成型，中央银行、证监会、银监会、保监会
  • 2009-2016：开始金融监管协调性和有效性初步改革
  • 2017年至今：一委一行两会新阶段，国务院金融稳定发展委员会、中国人民银行、中国银行保险监督管理委员会、中国证券监督管理委员会

国务院金融稳定发展委员会的职责

  1. 落实党中央、国务院关于金融工作的决策部署
  2. 审议金融业改革发展重大规划
  3. 统筹
  4. 分析
  5. 指导

中国人民银行的职责

  1. 拟定金融业改革、开放和发展规划；牵头国家金融安全工作协调机制，牵头负责系统性金融风险防范和应急处置；承担最后贷款人责任，负责对因化解金融风险而使用中央银行资金机构的行为进行检查监督。
  2. 牵头建立宏观审慎管理框架；制定和执行货币政策、信贷政策；负责制定和实施人民币汇率政策；负责国际国内金融市场跟踪监测和风险预警；持有、管理和经营国家外汇储备和黄金储备。
  3. 监督管理银行间债券市场、货币市场等；发行人民币，管理人民币流通；经理国库，管理国家外汇管理据，管理征信业，推动建立社会信用体系。
  4. 牵头负责重要金融基础设施建设规划并统筹实施监管；统筹金融业综合统计；承担全国反洗钱和反恐怖融资工作；

中国银行保险监督管理委员会的职责

  1. 依法依规对全国银行业和保险业实行统一监督管理；负责统一编制全国银行业和保险业监管数据报表。
  2. 参与起草银行业和保险业重要法律法规草案，制定审慎监管与行为监管规则，参加银行业和保险业国际组织与国际监管规则制定。
  3. 准入管理、从业人员行为管理规范。

中国证券监督管理委员会的职责

  1. 对证券期货市场实行集中统一监管。
  2. 依法对证券期货违法违规行为进行调查、处罚。
  3. 监管境内期货合约的上市、交易、清算，按规定监督境内机构从事境外期货业务。
  4. 管理有关证券公司的领导班子和领导成员，负责有关证券公司监事会的日常管理工作。
  5. 监管上市公司及其按法律法规必须履行有关义务的股东的证券市场行为
  6. 监管境内企业直接或间接到境外发行股票、上市；监管境内机构到境外设立证券机构；监管境外机构到境内设立证券机构、从事证券业务。
