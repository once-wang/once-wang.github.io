金融市场体系

发表于2022-07-27|更新于2022-08-04|证券从业资格证
|阅读量:

考题结构

  1. 金融市场体系：1-3分
  2. 中国的金融市场体系与多层次资本市场：8-13分
  3. 证券市场主体：13-15分
  4. 股票：15-18分
  5. 债券：15-18分
  6. 证券投资基金：15-18分
  7. 金融衍生工具：6-9分
  8. 金融风险管理：9-11分

金融市场概述

金融市场的概念与功能

金融市场的概念

  • 金融市场：金融市场是创造和交易金融资产的市场，是以金融资产为交易对象而形成的供求关系和交易机制的总和。
  • 金融市场是要素市场的一种，与产品市场不同。
  • 金融市场参与者不仅在有形的固定场所进行金融资产交易，还通过各种通信手段进行联系，形成庞大的无形金融市场网络，现代金融市场往往是无形的市场。有形市场和无形市场的界限是越来越模糊的。

金融市场的功能

自驾风流降成本

  1. 资金融通（首要功能）
  2. 价格发现
  3. 提供流动性：并不直接影响实体经济的资金供求
  4. 风险管理：保险、对冲交易等，金融衍生品
  5. 降低搜寻成本和信息成本：显性成本与隐性成本

直接融资与间接融资

直接融资与间接融资的区别：主要在于融资过程中资金的需求者与资金的供给者是否直接形成债权债务关系。

直接融资

直接融资的特点

不可自主查分值

  1. 直接性
  2. 融资的信誉有较大的差异性
  3. 分散性
  4. 部分融资方式具有不可逆性，股票
  5. 融资者有相对较强的自主性

直接融资的方式

古寨伤民风
证券市场融资（股票、债券）属于典型的直接融资方式。

  1. 股票市场融资
  2. 债券市场融资
  3. 风险投资融资，购买未上市企业发行的股权
  4. 商业信用融资
  5. 民间借贷

直接融资对金融市场的影响

  • 直接融资和间接融资比例反映了一国金融体系风险和分布情况；
  • 直接融资和间接融资比例反映了一国金融体系配置的效率是否与实体经济相匹配；提高直接融资比例
  • 通过市场主体充分博弈直接进行交易，更有利于合理引导资源配置，发挥市场筛选作用。

间接融资

间接融资的特点

  1. 资金获得的间接性
  2. 融资的相对集中性
  3. 融资信誉的差异性较小
  4. 全部具有可逆性
  5. 主动权掌握在金融中介机构手中

间接融资的方式

  1. 银行信用融资
  2. 消费信用融资
  3. 租赁融资

金融市场的重要性

  1. 促进储蓄-投资转化
  2. 优化资源配置
  3. 反映经济状态
  4. 宏观调控

金融市场的分类

按金融资产到期期限划分

  • 货币市场：一年以内
    • 银行间同业拆借视察过
    • 商业票据市场
    • 短期国库券市场
    • 大额可转让存单市场等
  • 资本市场：一年以上或者没有到期期限
    • 股票市场
    • 中长期国债市场
    • 中长期银行贷款市场等

按交易工具划分

  • 债权市场
  • 权益市场

按发行流通性质划分

  • 一级市场（发行市场、初级市场）
  • 二级市场（流通市场、次级市场）

按组织方式划分

  • 交易所市场（场内市场）
  • 场外交易市场

按交割方式划分

  • 现货市场
  • 衍生品市场：常见的衍生工具包括远期、期货、期权、互换等。

按金融资产的种类划分

  • 证券市场：以证券发行与交易的方式实现了筹资与投资的对接，有效地化解了资本的供求矛盾和资本结构调整的难题。
  • 非证券金融市场

按辐射地域划分

  • 国际金融市场
  • 国内金融市场

全球金融市场

全球金融市场与金融体系

全球金融市场的形成

  • 有明确实物和文字记载的最早的金融市场：古巴比伦乌尔城金融区
  • 出现证券交易活动最早的地区：16世纪的里昂、安特卫普（国债）
  • 最早的证券交易所：阿姆斯特丹证券交易所

全球金融市场发展的5个阶段

  1. 从17世纪英国崛起到第一次世界大战
    • 世界上第一家股份制商业银行：英格兰银行、伦敦交易所、英镑
  2. 从第一次世界大战开始到第二次世界大战结束
    • 美国；纽约市场成为最大的国际资本流动市场；美元
  3. 从第二次世界大战结束到1971年布雷顿森林体系瓦解
    • 纽约、伦敦和东京成为国际金融市场的“金三角”
    • 《关税及贸易总协定》
    • 《罗马条约》建立了欧洲经济共同体
  4. 从布雷顿森林体系瓦解到2007年全球金融危机前夕
    • 金融改革；全球性的宽松的金融监管、技术革新、金融创新和资产证券化。
  5. 2007年全球金融危机爆发以来：全球金融监管趋严

全球金融体系

全球金融体系的主要参与者

  • 住户
  • 金融公司：银行性和非银行性（证券、保险公司）
    • 存款吸收机构
    • 中央银行
    • 其他金融公司
  • 非金融公司
  • 公共部门
  • 广义政府
  • 为住户服务的非营利机构：不包括政府部门控制和提供资金支持的机构

国际资金流动方式

根据资金使用时间或交易期限的不同划分：

  1. 国际长期资金流动
    • 国际直接投资
      1. 采取独资企业、合资企业和合作企业等方式在国外建立一个新企业；
      2. 收购国外企业的股权达到拥有实际控制权的比例；
      3. 利润再投资
    • 国际间接投资
    • 国际信贷
      1. 政府信贷
      2. 金融机构贷款
      3. 国际银行贷款
      4. 出口信贷等
  2. 国际短期资金流动
    • 贸易资金的流动：最传统的国际资金流动方式
    • 套利性资金的流动：两个市场
    • 保值性资金的流动：避险性资金流动、资本外逃
    • 投机性资金的流动：同一个市场的不同时间段

国际金融监管体系

巴塞尔银行监管委员会

主要宗旨：

  1. 交换各国监管安排方面的信息
  2. 改善国际银行业务监管技术的有效性
  3. 建立资本充足率的最低标准
  4. 研究在其他领域确立标准的有效性

国际证监会组织（1983年成立）

主要宗旨：

  1. 致力于制定国际公认的监管准则和执法标准并推动其得到一致实施，以保护投资者，维护市场的公平、高效、透明，应对系统性风险；
  2. 通过加强在执法、市场及中介监管方面的信息交流和合作，加大投资者保护力度，增强投资者对证券市场诚信的信心；
  3. 为成员在全球和地区层面进行经验交流提供平台，以协助市场发展，推动市场基础设施建设，实施适度监管。

纲领性文件：《证券监管目标与原则》

  1. 保护投资者
  2. 确保市场的公平、高效和透明
  3. 减少系统性风险

国际保险监督官协会（1994年成立）

重要作用：

  1. 制定国际保险监管规则
  2. 发布国际保险最新动态
  3. 提供国际保险界交流平台

英国的金融市场

英国的金融市场类型

  1. 外汇市场：英国的伦敦外汇市场是全球最大的外汇批发市场。伦敦外汇市场的特点：
    • 以外币交易为主
    • 以非居民交易为主
    • 以金融机构批发性交易为主
    • 以外汇衍生品交易为主
    • 交易时段桥接美洲和亚太市场
  2. 银行间资金市场
  3. 股票市场：伦敦证交所的层次：主板市场、另类投资市场、专业证券市场、专业基金市场、精英项目。伦敦证交所的交易系统：SETS系统、SETSqx系统、SEAQ系统、IOB系统。
  4. 债券市场：金边债券市场、国际债券市场、公司债券市场
  5. 黄金市场：伦敦黄金市场是全球最大的黄金现货市场。
  6. 其他金融市场

英国的金融市场监管

  • 自律为基础的分业监管时期
  • FSA统一监管时期
  • 宏观审慎与微观行为监管并行的监管模式

美国的金融市场

美国的金融市场类型

  1. 外汇市场
  2. 银行间资金市场：由最贴近市场的纽约联邦储备银行负责根据联邦基金市场每日的交易情况，每日计算和发布成交额加权的有效联邦基金利率。
  3. 股票市场
    • 股票交易场所：全国性股票交易场所；另类交易系统；经纪商-交易商内部撮合
    • 纳斯达克集团：分层：全国、常规市场->精选、全球、资本市场。交易机制：混合交易模式（指令驱动+报价驱动）
    • 纽约证券交易所：指定做市商制度；场内经纪商制度；补充流动性提供商制度。
  4. 债券市场
    • 美国国债市场
    • 美国联邦机构债券市场
    • 美国市政债券市场（普通责任债券、收益债券、通道收益债券）
    • 美国公司债券市场
  5. 衍生品市场

指定做市商制度

做市商的职能：

  • 竞价的组织者
  • 经纪人职能
  • 稳定市场职能
  • 做市商职能

义务清单：

  • 正面义务：当市场存在或合理预期将存在缺乏价格连续性和深度、供需不平衡时，专家有义务以自营账户、自由资金逆市买进或卖出。
  • 负面义务：除非其交易经过合理的计算，有助于在合理的深度上维持价格连续性，以及使目前存在或合理预期将存在的临时性订单不平衡的影响最小化，否则专家不得进行自营买卖。

美国的金融市场监管

双层多头

  • 双层监管
    • 联邦、州
  • 分业监管
    • 银行业、证券业、保险业

中国香港金融市场

联系汇率制（与美元挂钩）

基础货币：

  • 已发行负债证明书（用作支持银行纸币）和由政府发行的纸币及硬币；
  • 持牌银行在香港金融管理局的结算账户结余总额；
  • 未偿还外汇基金票据和债券总额；

中国香港的金融市场类型

  1. 外汇市场：中国香港自2003年开放人民币业务以来，逐步成为全球人民币离岸市场中心。
  2. 股票市场：沪港通、深港通
  3. 债券市场：2017年，香港和内地之间启动了债券通计划，境内外机构和个人可以借助中国香港债券市场进入对方市场。

中国香港的金融市场监管

行业 金融监管机构 行业自律机构
银行业 香港金融管理局 香港银行公会
证券业 证券及期货事务监察委员会 香港交易所
保险业 保险业监理处 香港保险业联会
